## Project Setup Instructions


1. Clone the repository.

2. Install dependencies:
	`yarn install`

3. In the VS Code terminal, run:
	`npx expo`

4. The database is hosted on Supabase.
   - Physical Device: Scan the terminal QR code using the Expo Go app.
   - Android Emulator: Launch a virtual device in Android Studio, then press a in the terminal window

5. The database is hosted on Supabase.

6. Users / PW:
	- admin@example.com / admin
	- manager@example.com / manager
	- support@example.com / support
	- client@example.com / client
	
7. Start testing! 😊
